# Nvej
An Open Source, iOS 16+ muslim prayer times app.


## Features
- [x] Prayer times
- [x] Qibla direction
- [x] Hijri date
- [x] Prayer notifications
- [x] Prayer times widget
- [x] Prayer times widget configuration
